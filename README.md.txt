RevoLab Simulator v5.0

RevoLab is an advanced, browser-based industrial robotics simulator. It features real-time Inverse Kinematics (IK), workspace analysis, trajectory plotting, and AI-powered scripting via Google Gemini.

Features

Inverse Kinematics: Gradient descent solver for 6-DOF arms.

Visuals: High-fidelity 3D rendering with trails and ghost targets.

Analysis: 3D Convex Hull workspace generation and live trajectory plotting.

AI Copilot: Generate automation scripts using natural language.

Export: Generate C++ and Python driver code for your robot configuration.

How to Run

Ensure you have Python installed.

Clone this repository.

Run the launcher:

python launcher.py


The simulator will open in your default web browser.
